At Restlet we know about APIs. We provide tools for developers (and non-developers, too) to design, create, run and manage APIs that provide access to any data or application.

We recently launched APISpark, an innovative self-service platform to deploy, manage and use web APIs in cloud mode. APISpark deeply relies on open source Restlet Framework, first software for creating and using REST-style web APIs in Java. Restlet is also offering Restlet Studio, a lightweight web-based development environment that accelerates the crafting and design of APIs.
